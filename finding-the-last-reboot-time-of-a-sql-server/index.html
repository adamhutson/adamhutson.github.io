Finding The Last Reboot Time Of A Sql Server

Adam
2009-10-27  about 42 words  1 min 

Here is a quick way to find the last reboot date and time of a SQL Server instance.

For SQL Server 2000 or 2005, use: SELECT crdate FROM master.dbo.sysdatabases WHERE name = 'tempdb'

For SQL Server 2008, use: SELECT sqlserver_start_time FROM master.sys.dm_os_sys_info
